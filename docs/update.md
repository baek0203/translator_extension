# Instant Translate

Select text on any webpage, and translate it instantly.

## How to Use

1. **Select text** - Drag to select any text on a webpage
2. **Click the translate button** - A translation button appears next to your selection
3. **View translation** - See the result in a popup
4. **Save for later** - Save translations and revisit them anytime

## Features

- **Drag & Translate** - Select text anywhere and get instant translations
- **16 Languages Supported** - English, Korean, Japanese, Chinese, Spanish, French, German, Russian, Arabic, Portuguese, Italian, Vietnamese, Thai, Indonesian, Hindi, and more
- **Save Translations** - Keep your favorite translations for easy access
- **Vocabulary Dashboard** - View saved words and sentences in a two-column layout
- **Quick Navigation (Deeplink)** - Click saved translations to jump back to the original text on the page (Ctrl+F style highlighting)
- **Export** - Download your translation history as CSV
- **Font Size Control** - Adjust vocabulary text size directly on the page
- **Theme Support** - Light, Dark, or Auto (follows system)

## Dashboard Features

- **Vocabulary Tab** - Two-column layout separating Words & Phrases from Sentences
- **Select Mode** - Select multiple items for bulk deletion
- **Select All** - Quickly select all items for bulk operations
- **Font Size Control** - Adjust text size with -/M/+ buttons (S, M, L, XL)
- **Settings** - Configure translation language, UI language, and theme with Save/Cancel buttons
- **Information** - View extension version and contact support
